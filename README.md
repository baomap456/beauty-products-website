# Beauty Products Website

A full-stack beauty products website with Express.js backend and React frontend.

## Project Structure

```
beauty-products-website/
├── backend/          # Express.js API server
├── frontend/         # React web application
└── README.md         # This file
```

## Getting Started

### Backend (Express.js)

The backend is a REST API built with Express.js.

```bash
cd backend
npm install
npm start
```

Server runs on: `http://localhost:5000`

See [backend/README.md](backend/README.md) for more details.

### Frontend (React)

The frontend is a React application created with Create React App.

```bash
cd frontend
npm install
npm start
```

Application runs on: `http://localhost:3000`

See [frontend/README.md](frontend/README.md) for more details.

## Technologies

- **Backend**: Node.js, Express.js
- **Frontend**: React.js
