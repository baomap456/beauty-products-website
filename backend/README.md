# Backend - Express.js Server

This is the backend server for the Beauty Products website, built with Express.js.

## Getting Started

### Installation

```bash
cd backend
npm install
```

### Running the Server

```bash
npm start
```

The server will run on `http://localhost:5000`

### Available Endpoints

- `GET /` - Welcome message
- `GET /api/health` - Health check endpoint

## Development

To run the server in development mode:

```bash
npm run dev
```

## Technologies Used

- Node.js
- Express.js
